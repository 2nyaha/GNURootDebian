Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

In emulatorview:
* AndroidManifest.xml => emulatorview/src/main/AndroidManifest.xml
* res/ => emulatorview/src/main/res/
* src/ => emulatorview/src/main/java/
* src/jackpal/androidterm/emulatorview/package.html => emulatorview/src/main/resources/jackpal/androidterm/emulatorview/package.html
In Term:
* AndroidManifest.xml => Term/src/main/AndroidManifest.xml
* jni/ => Term/src/main/jni/
* lint.xml => Term/lint.xml
* res/ => Term/src/main/res/
* src/ => Term/src/main/java/
